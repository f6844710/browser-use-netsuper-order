# ブランドマッピング機能ガイド

## 概要
この機能により、ユーザーが商品名を言った際に、事前に登録したブランドを自動的に適用できます。

## 使い方

### 1. ブランドマッピングの登録
1. アプリケーションの右側にある「ブランド設定」パネルを探します
2. 「商品キーワード」フィールドに商品名のキーワードを入力（例：牛乳）
3. 「優先ブランド」フィールドにブランド名を入力（例：メグミルク）
4. 「追加/更新」ボタンをクリック

### 2. 商品の追加
商品を追加する際、登録されたブランドマッピングが自動的に適用されます。

**例：**
- キーワード「牛乳」にブランド「メグミルク」を登録
- ユーザーが「牛乳」と言うと、自動的に「牛乳（メグミルク）」として追加されます

### 3. マッピングの管理
- **一覧表示**: 登録されたマッピングは一覧に「商品キーワード → ブランド名」形式で表示されます
- **削除**: 一覧から削除したい項目を選択し、「選択を削除」ボタンをクリック
- **更新**: 既存のキーワードで再度登録すると、ブランド名が上書きされます

## 仕組み

### マッチングルール
1. **完全一致優先**: 商品名が登録キーワードと完全一致する場合
2. **部分一致**: 商品名に登録キーワードが含まれる場合
3. **大文字小文字を区別しない**: マッチングは大文字小文字を区別しません

### データ保存
- マッピングデータは `brand_map.json` ファイルに保存されます
- アプリケーション起動時に自動的に読み込まれます

## 使用例

### 例1: 乳製品
```
キーワード: 牛乳
ブランド: メグミルク
→ 「牛乳」を追加すると「牛乳（メグミルク）」になります
```

### 例2: パン
```
キーワード: パン
ブランド: Pasco
→ 「食パン」を追加すると「食パン（Pasco）」になります
```

### 例3: 飲料
```
キーワード: コーヒー
ブランド: UCC
→ 「アイスコーヒー」を追加すると「アイスコーヒー（UCC）」になります
```

## 注意事項
- 既にブランド名が商品名に含まれている場合は重複して追加されません
- マッピングは部分一致で動作するため、キーワードは具体的に設定することをお勧めします
- 複数のキーワードが一致する場合、完全一致が優先されます
